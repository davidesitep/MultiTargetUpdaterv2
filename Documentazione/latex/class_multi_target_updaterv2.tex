Utilizza Win\+SCP per la gestione dei file remoti e Open\+SSL per la crittografia e decrittografia dei dati. La classe include anche funzionalità per l\textquotesingle{}interazione con l\textquotesingle{}utente tramite un interfaccia gradica.

\doxysubsection{Detailed Description}
\doxylink{class_multi_target_updaterv2}{Multi\+Target\+Updaterv2} è una classe che gestisce l\textquotesingle{}aggiornamento di più dispositivi tramite un \textquotesingle{}interfaccia grafica. La classe permette di connettersi ai dispositivi, scaricare le configurazioni, verificare le versioni del firmware installato e disponibile, e gestire l\textquotesingle{}aggiornamento del firmware. Utilizza Win\+SCP per la gestione dei file remoti e Open\+SSL per la crittografia e decrittografia dei dati. La classe include anche funzionalità per l\textquotesingle{}interazione con l\textquotesingle{}utente tramite un interfaccia gradica. 

\begin{DoxyNote}{Note}
Questa classe è progettata in ambiente Qt e utilizza le librerie Open\+SSl per la criptografia e il client Win\+SCP per l\textquotesingle{}interazione con i dispositivi remoti e con il server FTP contenente i firmware. 
\end{DoxyNote}
